# 六方对弈：弹珠碰撞战

## 📖 游戏简介

《六方对弈：弹珠碰撞战》是一款基于**六边形棋盘**的回合制策略对战游戏。玩家操控六种拥有独特能力的弹珠，利用**弹性碰撞**与**角色特性**，将对方所有弹珠击出棋盘，即可获得胜利。

### ✨ 游戏特色

- **六边形棋盘**：169个格点，策略深度丰富
    
- **六种弹珠**：白、蓝、绿、红、黑、黄，各具独特机制
    
- **弹性碰撞系统**：真实速度交换，战斗节奏紧凑
    
- **死亡增益机制**：黄球牺牲可为友方提供永久强化
    
- **白球变色机制**：可继承死亡友方能力，战术灵活多变
    

---

## 🎮 游戏规则

### 胜利条件

将对方所有弹珠击出棋盘或使其死亡。

### 棋盘与坐标

- 正六边形区域，每条边8个格点
    
- 总格点数：**169个**
    
- 边界判定：`D = max(|x|, |y|, |x-y|) ≥ 8` 即出界
    

### 初始配置

- 红方（左） vs 蓝方（右）
    
- 每方拥有 **6个弹珠**，可从6种颜色中任意选取（可重复）
    
- 双方自行排布阵型，随机决定先后手
    

### 六种弹珠能力一览

| 颜色    | 定位    | 移动方式         | 特殊能力                    |
| ----- | ----- | ------------ | ----------------------- |
| ⚪ 白球  | 基础/继承 | 方向+力度1~5     | 被友方碰撞时+1剩余步数；友方其他色死亡时变色 |
| 🔵 蓝球 | 召唤/协同 | 方向+力度1~5     | 移动前生成2个随从；随从出界→蓝球死亡     |
| 🟢 绿球 | 推挤/控制 | 方向+力度1~5     | 移动后向远离方向推开相邻弹珠          |
| 🔴 红球 | 多段/灵活 | 步数1~4，每步可选方向 | 每步后立即碰撞并交换速度            |
| ⚫ 黑球  | 干扰/强制 | 不可主动移动       | 强制敌方弹珠移动（随机偏移±60°）      |
| 🟡 黄球 | 牺牲/增益 | 方向+力度1~5     | 死亡时强化友方其他弹珠             |

## 🛠 技术栈

- **引擎**：Godot 4.x
    
- **语言**：GDScript
    
- **架构**：策略模式 + 纯函数碰撞结算

## 🚀 快速开始

### 环境要求
    
- 支持 OpenGL 3 的显卡
    
### 运行步骤

在Releases中选择Tag标签，点击后选择最新版本下载即可。
### 操作说明

1. **点击**己方弹珠选中
    
2. 选择移动**方向**（六边形6个方向）
    
3. 选择移动**力度**（1~5步）
    
4. 点击执行，等待结算
    
5. 切换回合，对方行动
    

> 特殊弹珠（红球、黑球）操作方式略有不同，详见游戏内教程
